# Student Marks Analysis

## Project Overview

This project analyzes student performance data using Python, Pandas, and Matplotlib.

## Features

* Calculate average marks
* Find highest scorer
* Find lowest scorer
* Identify students scoring above 80
* Visualize marks using a bar chart

## Technologies Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Output

The project generates insights from student marks data and displays a graphical representation of student performance.
